# Solar Irradiance Forecasting using Hybrid XGBoost-LSTM Model

## Overview

This project forecasts solar irradiance using a hybrid machine learning and deep learning approach.

The model combines:

- XGBoost Regressor
- LSTM Neural Network

The objective is to predict future solar irradiance at different forecasting horizons:

- 1 Hour Ahead
- 6 Hours Ahead
- 12 Hours Ahead
- 24 Hours Ahead

## Dataset

Source: NASA POWER

Locations:

- Shimla
- Delhi
- Goa
- Jaisalmer

Period:

2019 - 2025

## Features

- ALLSKY_KT
- T2M
- RH2M
- PS
- WS10M
- PRECTOTCORR
- Hour
- Month
- DayOfYear

## Models

### XGBoost

Used as baseline model.

### Hybrid Model

XGBoost + LSTM Residual Learning

The LSTM learns residual errors generated by XGBoost and corrects future predictions.

## Evaluation Metrics

- MAE
- RMSE
- R² Score

## Results

Hybrid model achieved improved forecasting accuracy compared to standalone XGBoost.
